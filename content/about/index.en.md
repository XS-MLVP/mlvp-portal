This project aims to use open-source tools for the verification of open-source chips. This process not only means promoting the development of open source culture in the hardware field, but also provides opportunities and platforms for software developers to participate in hardware validation and development. In this project, we will explore and promote subcontracting verification methods, develop basic verification tools, and collect verification cases so that more software developers can participate in the hardware verification and development process.

We believe that such an open-source chip verification platform will bring more innovation and development opportunities to the entire industry, driving the hardware industry towards a more open and diversified direction.
