## About the Project
The Xiangshan Project is a high-performance RISC-V processor core open-source project based on Chisel. As a core infrastructure component, the current UT (Unit Test) verification module requires a sustainable developer ecosystem. We are now publicly recruiting the first batch of module maintainers to collaboratively enhance the verification framework. [Click here](https://www.wjx.top/vm/rxGeUKU.aspx#) to fill out the application form. After submission, join the QQ group (Group ID: <b>600480230</b>) to contact us.

---  

## Basic Requirements

UT module maintainers must possess a high-level understanding of at least one module to be maintained. For details on available modules, refer to the [Progress Overview](https://open-verify.cc/UnityChipForXiangShan/docs/). If you lack familiarity with Xiangshan’s current design but are strongly motivated to contribute to the open-source community, you may apply in advance to access module documentation and join formally after completing preparatory training.

## Core Responsibilities

### 1. Documentation Updates and Maintenance
Continuously update and maintain design documentation based on existing knowledge and evolving Xiangshan source code, ensuring accuracy in reflecting the latest module designs and verification processes to guide future developers.

### 2. Verification Framework Maintenance
Learn the [verification framework](https://github.com/XS-MLVP/UnityChipForXiangShan) used by the UnityChipForXiangShan project, optimize the verification environment, and improve module descriptions to ensure verification accuracy and efficiency.

### 3. Open-Source Crowdsourcing
Publish prepared modules on the "UnityChip" platform, lead crowdsourced verification efforts, and provide technical support.

## Incentives

Qualified participants will have the opportunity to be officially appointed as **Xiangshan UT Module Maintainers** and receive:

- **Maintainer Identity Badge**: GitHub account listed in the project’s maintainer roster.
- **Module Attribution Rights**: Verification modules you lead may be labeled "Maintained by {your_name}".
- **Community Contribution Certificate**: Electronic certificate issued by the Xiangshan core team.
- **Visibility in Open Source**: Outstanding contributors may be featured in official Xiangshan acknowledgments.
